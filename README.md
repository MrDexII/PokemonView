# Pokemon View 
Aplikacja kliencka do [Pokemon API](https://github.com/MrDexII/pokemonAPI), pozwala administratorom 
na zażądanie użytkownikami, oraz pokemonami, a zwykłym użytkownikom na wyświetlanie pokemonów.
